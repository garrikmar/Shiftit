# ShiftIT - מערכת לניהול והחלפת משמרות

## 🎯 סקירה כללית

ShiftIT היא מערכת מודרנית לניהול והחלפת משמרות עבור ארגונים B2B. המערכת מספקת פתרון מובנה להחלפת משמרות עם בדיקות אוטומטיות, ניהול אישורים, ומעקב אחר הוגנות בצוות.

## ✨ תכונות עיקריות

### 👥 עבור עובדים
- **לוח שנה אינטראקטיבי** - תצוגת שבוע/חודש/יום עם המשמרות
- **בקשת החלפה פשוטה** - תהליך מובנה עם בדיקות אוטומטיות
- **משמרות פתוחות** - צפייה ולקיחת משמרות זמינות
- **התראות בזמן אמת** - עדכונים על בקשות ואישורים
- **מעקב בקשות** - סטטוס ברור של כל בקשה

### 👔 עבור מנהלים
- **ניהול בקשות** - אישור/דחייה מהיר של בקשות החלפה
- **בדיקות Guardrails** - כשירות, כיסוי מינימלי, שעות מנוחה
- **ניהול צוות** - מעקב אחר חלוקת משמרות והוגנות
- **דשבורד מקיף** - תצוגה כוללת של המצב

## 🎨 עיצוב

- **Theme**: Dark Mode עם צבעי Emerald (Primary) ו-Blue (Secondary)
- **Direction**: RTL (עברית)
- **Font**: Inter
- **Responsive**: Mobile-first (390×844), Tablet (834×1112), Desktop (1440×900)

## 🧩 מבנה אפליקציה

### Components
- `ShiftCard` - כרטיס משמרת עם מצבים שונים
- `TopBar` - כותרת עם חיפוש והתראות
- `Sidebar` - תפריט ניווט ל-Desktop
- `BottomNav` - ניווט תחתון ל-Mobile
- `CalendarHeader` - כותרת לוח שנה עם בחירת תצוגה
- `FilterChips` - סינון משמרות
- `WelcomeBanner` - באנר ברכה
- `EmptyState` - מצב ריק

### Views
- `CalendarView` - לוח שנה (עמוד ראשי)
- `MyShiftsView` - המשמרות שלי
- `OpenShiftsView` - משמרות פתוחות
- `RequestsView` - בקשות שלי
- `ManagerPendingView` - בקשות ממתינות (מנהל)
- `TeamView` - ניהול צוות (מנהל)
- `NotificationsView` - התראות
- `SettingsView` - הגדרות

### Modals
- `SwapRequestModal` - תהליך בקשת החלפה (4 שלבים)
- `ManagerRequestDetailModal` - אישור/דחייה של בקשה

## 🔄 תהליך החלפת משמרת

1. **בחירת משמרת** - לחיצה על "לא יכול/ה להגיע"
2. **סוג פרסום** - בחירה בין פרסום לקבוצה או פנייה לאדם ספציפי
3. **פרטי בקשה** - הוספת סיבה (אופציונלי) ובחירת מחליף
4. **בדיקות אוטומטיות** - Guardrails (כשירות, כיסוי, מנוחה)
5. **שליחה** - הבקשה נשלחת לאישור/פרסום

## 📊 סטטוסי משמרות

- **מתוכנן** (Scheduled) - משמרת רגילה מתוכננת
- **ממתין לאישור** (Pending) - בקשה ממתינה
- **אושר** (Approved) - בקשה אושרה
- **נדחה** (Rejected) - בקשה נדחתה
- **פתוח** (Open) - משמרת זמינה לקבלה

## 💡 שימוש

### החלפת מצב (עובד/מנהל)
בקובץ `App.tsx`, שנה את השורה:
```typescript
const [isManager] = useState(true); // true למנהל, false לעובד
```

### נתוני דוגמה
האפליקציה מגיעה עם 10 משמרות לדוגמה הכוללות:
- משמרות בוקר/ערב/לילה
- תפקידים שונים (אחות, אח, אחות אחראית)
- מצבים שונים (מתוכנן, ממתין, פתוח, וכו')

## 📱 רספונסיביות

- **Mobile** (< 1024px): Bottom Navigation + Full width content
- **Tablet/Desktop** (≥ 1024px): Left Sidebar + Top Bar + Multi-column layout

## 🔔 התראות

המערכת משתמשת ב-Sonner לתצוגת Toast notifications:
- אישור בקשה
- דחיית בקשה
- לקיחת משמרת
- עדכונים כלליים

## 📝 הערות נוספות

- כל הטקסטים בעברית RTL
- אנימציות חלקות עם Tailwind transitions
- Dark theme בלבד (ברירת מחדל)
- Mobile-first approach
